### Dependencies

- Flask

## Setup

###### Windows

#### Make Repo

```
git clone repo_name
```

```
cd repo_name
```

```
py -3 -m venv venv
```

#### Activate Environment

```
venv\Scripts\activate
```

#### To deactivate enviroment

```
deactivate
```

#### Install Flask

```
pip install Flask
```

## Usage

#### Run App

```
flask --app ./main.py --debug run
```
