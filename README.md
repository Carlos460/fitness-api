### Dependencies

- Flask

## Setup

###### Windows

#### Make Repo

```
git clone repo_name
```

```
cd repo_name
```

### Create a virual enviroment

```
py -3 -m venv venv
```

Activate Script

```
venv\Scripts\activate
```

deactive virtual enviroment

```
deactivate
```

#### Install required packages

```
pip install -r ./requirements.txt
```

## Usage

#### Run App

```
flask --app ./main.py --debug run
```
